Теорема ферма

Для любого натурального числа n > 2 уравнение

a n + b n = c n

не имеет натуральных решений a, b, и c.

Язык C#

C# (произносится си-шарп) - обьектно ориентированый язык программирования. Разработан в 1998-2001 годах группой инженеров под руководством Андерса Хейлсберга в компании Microsoft как язык разработки приложений для платформы Microsoft .NET Framework и впоследствии был стандартизирован.

C# отсносится к семье языков с C-подобным синтаксисом, из них его синтаксис наиболее близок к C++ и Java. Язык имеет статическую типизацию, поддерживает полиморфизм, перегркузку операторов (в том числе операторов явного и неявного приведения типа), деленгаты, атрибуты , события, свойства, обобщенные типы и методы, итераторы, «анонимные» функции с поддержкой замыкания , LINQ, исключения, коментарии в формате XML.
